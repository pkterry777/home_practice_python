# 迴圈(階層加總) #

## 說明 ##

讓使用者輸入一個正整數n，求出1! + 2! + 3! + … + n!之值並輸出。<br>
n為大於零的數<br>
<br>
並且使用者輸入<b>浮點數(float)</b>或<b>英文字字串(string)</b>或<b>0</b>或<b>負數</b>時不會出錯，再要求使用者輸入直到大於零的正整數為止<br>
<br>
階層定義：<br>
0! = 1<br>
1! = 1<br>
2! = 1x2<br>
...<br>
5! = 1x2x3x4x5 <br>
...<br>
n! = 1*2*3*......*n <br>

## Input Format ##

整數 <br>

## Output Format ##

整數 <br>

## Sample Input 1 ##
```
10
```

## Sample Output 1 ##
```
4037913
```
## Sample Input 2 ##
```
NA
3
```
## Sample Output 2 ##
```
9
```
