# 讀取csv #

## 說明 ##

今天有一個班級的考試成績在<code>score.csv</code>中<br>
請開啟csv資料<br>
id：代表不同人<br>
math:代表數學分數<br>
eng:代表英文分數<br>


## Input Format ##

csv資料<br>

## Output Format ##

1. 列印出數學及格的人之ID(list)<br>
2. 列印出英文及格的人之ID(list)<br>
3. 列印出數學及格但英文不及格的人之ID(list)<br>
4. 列印出全班人數(int)<br>
<br>
*list的順序請先排序過再列印<br>

